## Step by Step Process:
# 1. **Create the Table**  
   - Make a table named `products`.  
   - Add three fields:  
     - `id` (auto-increment and primary key)  
     - `product_name` (text up to 100 characters, cannot be empty)  
     - `price` (decimal number)
